# ML-EmployeeAttritionPrediction
#### Employee Attrition Prediction using Machine Learning
### Dataset Overview
#### This project uses the IBM HR Analytics Employee Attrition & Performance dataset, which is designed to analyze and predict employee attrition.


#### The dataset is a tabular dataset for a binary classification task, where the target variable is:
Attrition:
Yes → employee leaves
No → employee stays


### Dataset Characteristics
Contains employee-level data
Includes both categorical and numerical features
Structured and clean, suitable for machine learning


### Key Features
Uncover the factors that lead to employee attrition and explore important questions such as ‘show me a breakdown of distance from home by job role and attrition’ or ‘compare average monthly income by education and attrition’. This is a fictional data set created by IBM data scientists.

## Why this dataset
This dataset represents a realistic HR problem where organizations aim to reduce employee turnover. Predicting attrition allows companies to:
identify high-risk employees
reduce hiring and training costs
improve workforce stability
This makes it suitable for building a machine learning model that supports decision-making rather than only performing prediction.
